## ✨ Features
- **Tiered Membership**: Integrated membership system via *Paid Member Subscriptions* that restricts premium articles and podcasts to paid subscribers.
- **Podcast & Media Hub**: Designed to showcase audio and video content for a journalism-focused audience.
- **Minimalist Aesthetic**: Built with the *Nothing Theme* for an ultra-clean, functional, and distraction-free reading experience.
- **Visual Page Building**: Utilizes *Elementor* for flexible and dynamic layout construction.

## 🛠️ Tech Stack
- **Core**: [WordPress 6.9.4](https://wordpress.org/)
- **Theme**: [Nothing Theme](https://github.com/mDev/nothing-theme) (Minimalist Functional Theme)
- **Page Builder**: [Elementor](https://elementor.com/)
- **Membership**: [Paid Member Subscriptions](https://www.cozmoslabs.com/wordpress-paid-member-subscriptions/)
- **Environment**: PHP, MySQL

## 🚀 Getting Started

### Prerequisites
- PHP 7.4 or higher
- MySQL 5.7 or MariaDB 10.3 or higher

### Installation
1. Clone the repository to your local web server (e.g., XAMPP/LAMPP `htdocs`).
2. Create a MySQL database (suggested name: `wp_vale_journal`).
3. Configure your `wp-config.php` with your database credentials.
4. Ensure the `wp-content/uploads` directory is writable by the web server.
